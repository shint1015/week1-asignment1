# Student Manager (UI + Modals)

Small app to add students, save to localStorage, and browse API users with smooth, animated modals.

## What it does

-   Add student (Name + Grade) with validation and de-duplication
-   Save list to localStorage and show feedback in an app modal
-   Fetch users (JSONPlaceholder); click a card to view a detail modal

## ES6+ used

-   const/let, arrow functions, template literals
-   Set/Map, spread, Object.fromEntries, nullish coalescing (??)

## Run

Open `index.html` in a browser (or a simple static server like VS Code Live Server).
